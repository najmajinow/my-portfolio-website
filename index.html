About Me

Hi there! I'm a web developer who loves creating beautiful and easy-to-use websites. I've always been fascinated by how websites work, and over time, I taught myself how to build them using HTML, CSS, and JavaScript. Now, I specialize in making static websites that look great and do exactly what my clients need them to do.

I pay a lot of attention to detail and always aim to write clean and efficient code. I'm also big on communication—I want to make sure that every project I work on perfectly matches what the client has in mind.
